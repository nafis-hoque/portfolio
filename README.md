## ğŸš€ Getting Started

### Prerequisites

- A modern web browser
- (Optional) A local server for better development experience

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/nafis-hoque/nafis-hoque.git
   cd nafis-hoque
   ```

2. Open `index.html` in your browser, or serve it locally:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .

   # Or any other static file server
   ```

3. Navigate to `http://localhost:8000` (or your chosen port)
